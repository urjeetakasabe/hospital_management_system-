Hospital Management System

A web-based Hospital Management System developed using Java Spring Boot, Spring Security, Hibernate/JPA, H2 Database, and Thymeleaf.

# Features

* Patient Registration and Login
* Secure Authentication with Spring Security
* Appointment Scheduling System
* Patient Dashboard
* Calendar-Based Appointment Tracking
* Admin and Patient Role Management

# Technologies Used:

- Backend

* Java 21
* Spring Boot
* Spring Security
* Spring Data JPA
* Hibernate

- Database

* H2 Database
* SQL

- Frontend

* Thymeleaf
* HTML
* CSS
* Bootstrap

- Tools

* Maven
* Git
* GitHub
* IntelliJ IDEA

# Project Architecture

Controller → Service → Repository → Database

# How to Run

1. Clone the repository
2. Open in IntelliJ IDEA
3. Run HospitalApplication.java
4. Open http://localhost:8080
